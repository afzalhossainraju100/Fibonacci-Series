Simple Fibonacci script

- File: `fib.py`
- Purpose: Print the first N Fibonacci numbers in a clear, easy way.

Usage (Windows `cmd.exe`):

```
python fib.py 10
```

or run and follow the prompt:

```
python fib.py
# then enter a number when asked
```

The function `fibonacci(n)` returns a list of the first `n` Fibonacci numbers.
